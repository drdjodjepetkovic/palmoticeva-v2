# **App Name**: AccessMatrix

## Core Features:

- Google Sign-In: User authentication via Google Sign-In, storing user IDs in Firestore.
- Role-Based Access: User role-based access control to different pages (Home, Menstrual Calendar, My Profile).
- Modular Data Structure: Modular data structure: each element will have a unique ID stored in Firestore along with associated data, including translations (SE, EN, RU).
- Home Page: Home Page access for all users.
- Menstrual Calendar: Menstrual Calendar accessible to logged-in users.
- My Profile: My Profile page accessible to externally verified users.

## Style Guidelines:

- Primary color: HSL 210, 75%, 50% (approximately #2699F0) for a trustworthy and accessible feel, referencing the app's central management of permissions.
- Background color: HSL 210, 20%, 98% (approximately #FAFCFF), providing a light and unobtrusive backdrop.
- Accent color: HSL 180, 75%, 50% (approximately #26F0C4) to highlight key interactive elements and CTAs.
- Body and headline font: 'Inter' sans-serif, for a modern and neutral look suitable for UI elements.
- Simple, geometric icons for navigation and data representation.
- Clean, card-based layout for modular content and clear information hierarchy.
